# PASSWDB


## description:
	"" This is a 'passwdb' app for upkeep a passwords. ""

## used libraries:

	1- slqite3
	2- hashlib
	3- colorama
	4- termcolor
	5- readline
	6- os, sys
	
## how to run:

	pip install termcolor
	pip install colorama
		OR:
	python3 requirements.py
		THEN:
	python3 PASSWDB.py
